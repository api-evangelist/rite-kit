# Rite KitThis is a repo for managing machine readable artifacts for the [Rite Kit APIs](http://ritekit.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Rite Kit APIs](http://ritekit.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://ritekit.com)- [Developer](http://ritekit.com)- [Twitter](https://twitter.com/RiteTag)- [GitHub](https://github.com/RiteKit)The [APIs.json](https://github.com/api-evangelist/rite-kit/blob/master/apis.json) index provides links to all the relevant links for the Rite Kit APIs, including the OpenAPI and Postman if they are present.
